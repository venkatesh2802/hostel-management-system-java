\# Hostel Management System (Java)



A console-based Java application to manage hostel operations such as room allocation, member records, and payments.



\## Features

\- Admin authentication

\- Room allocation

\- Member management

\- Payment tracking

\- File-based data storage



\## Technologies

\- Java

\- File Handling

\- OOP



\## Run the project

javac JavaMin.java

java JavaMin
